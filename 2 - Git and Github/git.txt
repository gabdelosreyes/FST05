Git commands:

git init - Initialize a repository
git --version - check ver of git
git status - check status of Git
git add . - stage all of the files/directories
git add file_name/foldername - stage specific file/directory
git commit -m "description" - commit files into repository
git log - check the recent commit logs
git log --oneline - check recent commited logs in one line
git config user.name "username" - configure username
git config user.email "email" - configure email
git config user.* - check *username or *email
git commit --amend -m "changed description" - rename the recent committed repository

Legends
U - Untracmked
A - Added
M - Modified
